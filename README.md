# 🔴 FOCUSNODE

**FocusNode** is a futuristic, AI-enhanced Pomodoro timer designed for high-performance productivity. It blends a minimalist cyberpunk aesthetic with intelligent neural heuristics to help you achieve a flow state and stay synchronized across all your devices.

![FocusNode Architecture](https://images.unsplash.com/photo-1550751827-4bd374c3f58b?auto=format&fit=crop&q=80&w=1200&h=400)

## ⚡ Core Heuristics

- **Intelligent Uplink**: AI-powered productivity insights generated by the Gemini neural network.
- **Node Synchronization**: Synchronize your focus history across multiple devices using a unique Identity Key—no account required.
- **Hourly Streak Analysis**: Visualize your most productive hours with our integrated cluster dashboard.
- **Neural Overrides**: Master your workflow with professional keyboard shortcuts.
- **Glassmorphic UI**: A high-contrast, distraction-free interface built for focus.

## 🛠 Neural Stack

- **Frontend**: React 18 + Vite (TypeScript)
- **Styling**: Tailwind CSS (Glassmorphism & Cyberpunk accents)
- **Database**: Better-SQLite3 (Local persistent data clusters)
- **AI Engine**: Google Gemini API (Strategic productivity tips)
- **Motion**: Framer Motion for organic state transitions

## ⌨ Neural Overrides (Keyboard Shortcuts)

Boost your efficiency by staying on the keyboard:

| Key | Action |
| --- | --- |
| `P` / `Space` | Toggle Timer (Flow/Pause) |
| `R` | Reset Current Sequence |
| `W` | Activate Focus Mode (Work) |
| `S` | Activate Rest Phase (Short Break) |
| `L` | Activate Uplink Phase (Long Break) |

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd focusnode
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment**:
   Create a `.env` file from `.env.example`:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   JWT_SECRET=your_jwt_secret_here
   ```

4. **Initialize Core**:
   ```bash
   npm run dev
   ```

## 🌌 Architectural Philosophy

*Privacy first. Performance always.* FocusNode stores your data locally and uses unique synchronization keys to bridge the gap between your machines. No centralized trackers, just pure focus logic.
